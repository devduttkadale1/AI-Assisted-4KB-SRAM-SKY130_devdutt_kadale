* Studied logical and physical memory organization used by OpenRAM.

### Key Learnings

#### Precharge Circuit

* Bitlines are precharged before every read operation.
* Precharging ensures both BL and BLB start from the same voltage level.
* This improves sensing accuracy and read reliability.

#### Sense Amplifier

* Sense amplifiers detect very small voltage differences between BL and BLB.
* They convert analog voltage differences into full digital logic levels.
* Sense amplifiers are essential for high-speed SRAM operation.

#### OpenRAM Configuration

* OpenRAM uses configuration files to define memory architecture.
* Key parameters include:

  * `word_size`
  * `num_words`
  * `tech_name`
  * `process_corners`
  * `supply_voltages`
* A memory specification can be converted directly into a physical SRAM macro.

#### Memory Size Calculation

For the reference design:


Address width:

```text
log2(1024)
```

#### OpenRAM Flow

```text
Memory Specification
→ Configuration File
→ SKY130 Technology Files
→ Custom Cells
→ OpenRAM Compiler
→ Characterization
→ Generated Outputs
```

#### Logical vs Physical Organization

Logical memory:

```text
1024 × 32
```

OpenRAM physical organization:

```text
128 Rows
256 Columns
8 Words per Row
```

This organization reduces bitline length, improves speed, and reduces capacitance.
