# AI-Assisted Engineering Workflow

## Objective

Use AI tools to accelerate understanding of SRAM architecture, OpenRAM flow, and SKY130 technology integration while independently verifying all findings.

## Workflow

```text
Technical Question
        ↓
AI Prompt Creation
        ↓
AI Response Generation
        ↓
Repository Verification
        ↓
Documentation Creation
        ↓
Engineering Notes
```

## Tools Used

### ChatGPT

Used for:

* SRAM architecture study
* 6T SRAM cell analysis
* Read/write operation understanding
* OpenRAM flow explanation
* SKY130 technology exploration
* IEEE report preparation

### Codex

Used for:

* Repository exploration
* Configuration analysis
* OpenRAM output analysis
* Documentation generation support

## Verification Method

All AI-generated explanations were verified against:

* SRAM_SKY130 repository
* OpenRAM documentation
* SKY130 technology files
* Technical textbooks and reference material

## Key Principle

AI was used as a learning accelerator rather than a replacement for engineering understanding.
