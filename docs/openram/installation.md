# OpenRAM Installation Summary

## Environment

- Ubuntu 24.04 (WSL2)
- Python 3.12
- SKY130A Open-Source PDK
- OpenRAM v1.2.49

## Installation Steps

- Cloned OpenRAM stable branch.
- Created Python virtual environment.
- Installed required Python packages.
- Configured environment variables.
- Verified compiler execution.
- Created SRAM configuration.
- Executed OpenRAM compiler.

## Notes

The OpenRAM compiler executed successfully but enforced its minimum supported SRAM organization.

The compiler source was intentionally left unmodified to preserve the standard OpenRAM flow.
